Ventaroot - Static website
==========================
Contenu: index.html, histoire.html, membres.html, cheats.html, status.html, style.css, script.js, assets/*

Déployer sur Netlify:
1. Téléversez le dossier 'ventaroot_site' sur GitHub ou glissez-déposez le zip depuis Netlify "Deploy from ZIP".
2. Point d'entrée: index.html
3. Pas de build nécessaire (site statique).

Personnalisation:
- Remplacez les pistes dans /assets (track1.mp3, track2.mp3).
- Remplacez logo.png par votre logo (même nom).
- Modifiez les textes dans les .html.

Fichiers inclus:
 - index.html
 - histoire.html
 - membres.html
 - cheats.html
 - status.html
 - style.css
 - script.js
 - assets/logo.png
 - assets/track1.mp3
 - assets/track2.mp3
